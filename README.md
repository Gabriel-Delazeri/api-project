Projeto Pokedéx desenvolvido utizando Laravel (PHP) -- Última atualização (10/03/2022)

Comandos: 

- composer install
- npm install
- npm run dev
- php artisan key:generate (talvez)
- php artisan migrate
- php artisan db:seed --class=PokemonSeeder (vai popular o banco de dados com dados da api)
